# Diamond Dust Drinks — Sitio Web Interactivo

**Diamond Dust Drinks** es una página web moderna y responsiva creada con **HTML5**, **CSS3** y **JavaScript**, diseñada para mostrar la carta digital de un bar artesanal con cócteles, cervezas y maridajes.  
El sitio combina un diseño elegante con efectos visuales vibrantes e interactividad básica mediante scripts de JavaScript.

---

## Características principales

- **Diseño responsivo:** Se adapta automáticamente a dispositivos móviles, tablets y escritorios.
- **Modo claro/oscuro:** Alternancia visual mediante un interruptor de tema.
- **Paleta de colores neón:** Inspirada en un bar nocturno, con tonos brillantes (naranja, lima, fucsia y dorado).
- **Interactividad JavaScript:**
  - Diálogos de bienvenida (`alert` y `prompt`).
  - Validación de edad antes de ingresar.
  - Cálculo dinámico del total de consumo.
  - Bucles (`while`, `for`), operadores (`==`, `>`, `<`, `++`) y condicionales (`if`, `else if`, `break`).
- **Animaciones suaves:** Transiciones y efectos de brillo al pasar el cursor sobre botones o tarjetas.

---

## Funcionalidades JavaScript implementadas

- **Verificación de edad:**  
  Solicita la edad del visitante y muestra un mensaje dependiendo de si es mayor o menor de edad.

- **Contador de visitas:**  
  Un bucle `while` con incremento (`visitas++`) y `break` para limitar la cantidad de mensajes.

- **Cálculo del total del pedido:**  
  Usa una función que multiplica cantidad \* precio y muestra el total mediante `alert()`.

- **Creación dinámica de elementos:**  
  Inserta un botón “Calcular total” en la sección de contacto usando `createElement()` y `appendChild()`.

- **Mensajes en consola:**  
  Un bucle `for` imprime promociones simuladas en la consola del navegador.
